# `@feugene/unocss-preset-granular` — документация

Универсальный UnoCSS‑пресет, который агрегирует стили, темы и `safelist` из
произвольного числа **granular‑провайдеров** (пакетов компонентов). Пресет
не знает ни про один конкретный UI‑пакет — он работает поверх публичного
контракта `GranularProvider`. Решения о наборе компонентов и тем принимает
конечное приложение в своём `uno.config.ts`; провайдеры лишь декларируют,
что они поставляют.

> 🇬🇧 English version: [`../en/README.md`](../en/README.md).

## Кому это

- **Авторам приложений** — если хочется подключать компоненты из одного или
  нескольких пакетов‑провайдеров, не поддерживая руками `safelist`, не
  переопределяя CSS‑токены и не таща в сборку классы не использованных
  компонентов. → [Использование в приложениях](./usage-in-apps.md).
- **Авторам пакетов компонентов** — если вы поддерживаете пакет UI‑компонентов
  и хотите выставить его как полноценного провайдера (safelist, зависимости,
  темы, токены) для любого downstream‑приложения на UnoCSS. →
  [Написание провайдеров](./authoring-providers.md).

## Что делает пресет, в одном абзаце

Получив список `providers` (пакетов, которые экспортируют объект
`GranularProvider`) и список `components`, которые приложение реально
использует, пресет:

1. резолвит транзитивный граф `dependencies` выбранных компонентов (в том
   числе cross‑provider рёбра);
2. эмитит нужный `safelist` (только те классы, которые собираются в
   рантайме и не извлекаются статически);
3. на build‑time (`/node`) читает с диска нужные CSS‑файлы (base, tokens,
   per‑theme, per‑component) и заворачивает их в UnoCSS `preflights`;
4. сообщает UnoCSS **какие исходники выбранных компонентов сканировать**
   через `content.filesystem` — чтобы статические `class="p-5"` попадали
   в итоговый CSS автоматически;
5. мержит опциональные `rules` / `variants` / `preflights`, которые
   приносит провайдер в своей секции `unocss`.

Итог: **шаблон компонента — единственный источник правды** по его классам;
приложение не дублирует их в `safelist`, пресет не тащит лишние компоненты.
